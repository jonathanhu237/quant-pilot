## Project Overview

This project is an AI-powered mobile app for quantitative strategy research and paper trading.  
It is built with **React Native** for the frontend and **Python + FastAPI** for the backend.

The app does **not** support real trading or brokerage integration.  
Its main goal is to help users understand quantitative strategies, monitor market signals, simulate trades, and review performance with the help of **LLM + RAG**.

## Core Features

- **Home Dashboard**  
  Shows strategy highlights, recent signals, paper trading performance, and daily AI summaries.

- **Market Page**  
  Displays watchlists, stock details, price charts, technical indicators, and strategy signal markers.

- **Strategy Center**  
  Provides strategy descriptions, use cases, parameter settings, and backtest results.

- **Paper Trading**  
  Allows users to simulate buy/sell actions with virtual capital, and track positions, returns, and trading history.

- **AI Assistant**  
  Supports natural language Q&A for strategies, reports, portfolio performance, and daily trading review.

## LLM + RAG Capabilities

- Explain strategy logic and technical indicators in plain language
- Answer questions based on uploaded research reports, PDFs, and strategy documents
- Generate daily or periodic performance summaries
- Explain why a strategy generated a signal or why a portfolio gained/lost value

## Scope Boundaries

- No real-money trading
- No live brokerage account connection
- No automatic order execution
- No personalized financial advice
